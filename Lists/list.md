# List

## A list is a mutable, ordered collection of items.

### Common Operations:

- Append: list.append(x)

- Extend: list.extend(iterable)

- Insert: list.insert(i, x)

- Remove: list.remove(x)

- Pop: list.pop(i)

- Index: list.index(x)

- Count: list.count(x)

- Sort: list.sort()

- Reverse: list.reverse()
